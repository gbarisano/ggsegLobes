# ggsegLobes: Lobar Parcellation Atlas for the 'ggsegverse' Ecosystem

Lobar cortical parcellation (frontal, parietal, temporal, occipital,
insula, per hemisphere) derived from the Desikan-Killiany atlas by
dissolving its cortical regions into lobes. Provided as a 'ggseg_atlas'
object for use with 'ggseg'.
